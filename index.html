SNT : Le web

Bienvenue sur mon site de SNT spécial Web. Dans cette première séance, nous allons poser les bases de notre parcours et découvrir les fondamentaux du web et de l'informatique. À partir de la Séance 2, nous commencerons à aborder les aspects pratiques du codage en HTML, avec pour objectif final de créer et de structurer une page web complète lors des séances 3 et 4. Ces séances vont nous permettre de comprendre comment le web fonctionne, de découvrir les outils et langages qui le composent, et de développer les compétences nécessaires pour concevoir un site web de A à Z.
